# docker-flask-psql-app

The project is a Dockerized Flask application with PostgreSQL integration, providing a ready-to-run environment for developing and deploying web apps with Python and PostgreSQL.

            
 ## How to send a POST request with command line       

curl -X POST <url/add_data_post> \
     -H "Content-Type: application/json" \
     -d '{"city": "Oslo", "temperature": 19}'
